DATA-TREE
An Angular component for rendering a hierarchical table
  • You can collapse or expand the tree nodes can be by clicking on the +/- icon. This is a toggle button.
  • You can also collapse or expand all child nodes by pressing the Alt or Ctrl key when you click on the toggle button.
  • Data for the tree must be of TreeNodeType. The groupKey property defines the text that is used by the tree node. The subGroups property defines the child nodes. subGroups must be an array of TreeNodeType or null;
  • You can add icons to nodes by a specifying an iconClass.
  • You can combine a data-table together with a data-tree to show a tree table.
