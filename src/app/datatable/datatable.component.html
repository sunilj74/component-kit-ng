DATA-TABLE
An Angular component for rendering an advanced table
  • You can enable sort on columns. Users will then be able to sort in an ascending or descending order. Users are also able to perform multi-column sorts. An additional sort column can be included by ctrl-clicking on the sort button.
  • You can also perform custom sorting by listening to sortOrderChanged event. This will ignore the default sorting implemented in the data-table
  • You can enable resize on columns. Users will then be able to resize columns by dragging the column grid lines.
  • You can specify a group header for related columns.
  • When you have a large number of rows in your table, you can specify a page size. This will let you present data more neatly. Users will then be able to go to the different pages. First page, last page, previous and next page shortcuts are available. Users can also specify a page no to go to that page directly.
  • Support for hierarchical data. If your table data have nested table data you can use the children option to define a child grid for the child colllections. All data will then be displayed like a tree table. You can enable a collapse option for the children tables. Users will then be able to hide or show child data.
  • Floating sections can also be added at row-level. This is a alternate way to prevent horizontal scrollbars.
  • If you are dealing with a large collection of data and do not want to fetch the entire row set upfront, you can fetch the data in buffers. You can then listen to page navigation to fetch additional data.
