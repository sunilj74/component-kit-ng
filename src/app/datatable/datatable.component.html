<data-table> is an Angular component for rendering an advanced table or a data-grid. It has some features that users might find very useful.
  • sort on one or more columns
  • sort ascending or descending
  • resize columns
  • group columns
  • pagination
  • users can jump to any page
  • support for child colllections
  • floating section at row-level
  • support for buffered data fetching
